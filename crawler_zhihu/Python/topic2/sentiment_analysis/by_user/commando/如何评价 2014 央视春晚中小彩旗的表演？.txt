如何评价 2014 央视春晚中小彩旗的表演？

是丧心病狂，还是超乎想象？扩展问题：如何定义一个表演的意义所在？如果观众产生不可抑制的同情是会给这个作品加分，还是减分？

就算我了解她能够，且喜欢自己转4个小时圈儿，我也觉得在春晚舞台上这么转毫无意义。就我个人而言，她在我看不到的地方转圈（这和舞剧不同，转4个多小时，在电视屏幕上出现不到60秒而已）丝毫没有给我带来任何愉悦，反而给我不时带来焦虑。把场景换成一个人绕着演播大厅跑圈儿，或者一个大只佬不停地深蹲，或者一个杂技演员不停转盘子，或者一个大汉不停吃汉堡，或者600个大娘连续跳广场舞破吉尼斯记录……我看也都差不多。抛开残忍不残忍（但我真的觉得这很残忍）不提，关键是我不太了解这么转圈儿有什么意义……
