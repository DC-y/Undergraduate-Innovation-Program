chrome_js
=====================

这是一个基于Javascript的用于抓取知乎用户followers、followees的用户信息的工具。

                              `2014-01-23`

这是一个用js写成的，基于chrome的js调试控制台运行的爬虫。运行时只需(在Mac平台)`command+option+j` 然后将代码复制进去并回车；或者(Windows平台) `F12` 亦如此。
